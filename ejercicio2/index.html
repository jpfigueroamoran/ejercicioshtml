Funciones y etiquetas

Función:
Es una porción de código que se define una sola vez pero se puede ejecutar múltiples veces a lo largo del programa. Su objetivo es modularizar el código para hacerlo más organizado y reutilizable.
let:
Es una palabra clave de JavaScript que se utiliza para declarar variables cuyo valor puede cambiar a lo largo del programa.
const:
Es una palabra clave de JavaScript que se utiliza para declarar variables cuyo valor no puede cambiar a lo largo del programa.
Etiqueta blockquote:
Es una etiqueta de HTML que se utiliza para citar un fragmento de texto de otro autor. El texto citado aparece con un sangrado y en cursiva.
Etiqueta abbr:
Es una etiqueta de HTML que se utiliza para indicar una abreviatura o acrónimo. Si se sitúa el cursor sobre la abreviatura, aparece el término completo.
Etiqueta mark:
Es una etiqueta de HTML que se utiliza para resaltar un fragmento de texto. El texto resaltado aparece con un color de fondo amarillo por defecto.
